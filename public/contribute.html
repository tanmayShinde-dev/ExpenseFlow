How to Contribute

Follow these simple steps to add your profile card and make your first open-source contribution üöÄ

01

Prepare Your Image

  • Square image (1:1 ratio)
  • jpg or png format
  • Under 500KB
  • Name it as your GitHub username
02

Fork & Clone

Fork the repository to your GitHub account, then clone it to your local machine:

            git clone https://github.com/your-username/repo-name.git
03

Add Your Details

Upload your image to images/ and update index.html with your card details.

04

Commit & Create PR

                    git add .
                    git commit -m "Add my profile card"
                    git push origin main

Create a Pull Request to the main repository.
